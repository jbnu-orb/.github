# ORB(Object Remover for Blogger)

## About

안녕하세요 👋, 전북대학교 **"2024 생성 AI를 활용한 서비스 소프트웨어 개발 공모전의 ORB"** 팀 프로젝트 저장소입니다.

> **블로거를 위한 객체 제거 서비스**
> - 웹사이트에서 원하는 부분 페인팅시 제거
> - 이미지에 알맞는 캡션도 생성
> - 이미지 편집 기능은 SNS 포스팅, 프로필 사진 등 다양하게 활용 가능
